# 古诗源 · 领域上下文

本文件是站点的单一领域上下文（single context），记录关键领域词汇与既定决策。决策细节见 `docs/adr/`。

## 底本（base edition）

**权威底本：中华书局点校本《古诗源》。**

- 「古诗源」指清·沈德潜选评的具体选本（十四卷，录诗七百余首），不是「唐前古诗」的泛称。
- 卷目、选目、异文校勘一律以中华书局点校本为准（见 ADR 0001）。
- 内容以**繁体**录入为权威字形；简体由构建期 `t2s` 派生（见 #22）。

## 古逸（guyi）

《古诗源》卷一，收上古至先秦的逸诗、歌谣、谚语（击壤歌、越人歌、易水歌、卿云歌等）。是原书目录的**第一条**，体现沈德潜「《诗经》《楚辞》之外、上探唐诗之源」的编选立意。站点 `volumes.json` 首卷 `slug: gu-yi`。

## 卷（volume）

站点按原书朝代段归并分卷，共 10 段：古逸 / 汉 / 魏 / 晋 / 宋 / 齐 / 梁 / 陈 / 北朝 / 隋（对应原书十四卷）。路由 `/v/[volumeSlug]`。卷下按诗人 → 诗 → 阅读页。

## 源流（lineage）

源流讲《古诗源》**选诗**如何被后世化用。每条线索（clue）指向一首选诗的某句，并列出后世承袭的「流」（streams）。三种关系：`化用` / `脱胎` / `意象承接`。诗经楚辞不在选目内，故不作为源流源头。

## 阅读方向（reading direction）

横排（默认）/ 竖排（列自右向左）。偏好存 localStorage，key `gushiyuan-reading-direction`。

## 字形演变（已弃用）

字形演变功能（`/c/[char]`、`content/characters/*`）将整体移除（见 #18），其点击 Popover 模式迁用于「源」注解。移除后「源」为唯一行内注解形态。
